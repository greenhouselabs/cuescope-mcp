# Script Examples

Examples are intentionally review-first. They show safe patterns for generated artifacts, not instructions to execute blindly.

Included examples:

- `video-end-trigger.md`: one-shot media-end trigger with XML polling.
- `lower-third-timed.md`: update and show a lower third, then hide it.
- `real-world/`: sanitized working-script corpus guidance for known-good production scripts.
