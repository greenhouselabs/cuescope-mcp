# Virtual Set Host Guest Patterns

Host/guest virtual sets often combine:

- Host camera.
- Guest call or browser input.
- Background or set graphic.
- Lower thirds and bugs as overlays.
- Audio routing for host, guest, and return feeds.

Review Mode analysis:

- Identify likely host and guest inputs from production roles.
- Identify scene or virtual-set inputs that layer those sources.
- Explain which layers are probably source windows.
- Recommend visual verification before changing pan/zoom/crop.

Generation guidance:

- Return setup and rehearsal steps with any layout plan.
- Use layer numbers and stable keys where available.
- Avoid executing layout edits in Review Mode.
