# Bus Routing Recipes

Basic podcast:

- Host mic: `M,A`
- Guest mic: `M,A`
- Music bed: `M`
- Comms: `B`

Remote guest mix-minus:

- Guest input: `M`
- Host input: `M,A`
- Playback: `M,A`
- Guest return output uses bus `A`.

Green room:

- Program audio to green room bus.
- Guest mics not sent back to themselves.
- Producer talkback sent only when intentional.

Review checklist:

- Confirm bus letters match the output routing in vMix settings.
- Confirm muted state separately.
- Confirm embedded audio on videos is routed intentionally.
